# AX Marketing Group Bot

Telegram-бот для розсилки повідомлень в одну групу. Адмін надсилає повідомлення, бачить превью і обирає — відправити зараз або запланувати на конкретний час (київський часовий пояс).

## Функціонал

- Відправка повідомлення в групу одразу
- Планування на +30 хв / +1 год / +2 год або на конкретний час (ГГ:ХХ)
- Перегляд і скасування запланованих розсилок
- Підтримка: текст, фото, відео, документ, аудіо, голосове, кружок

## Команди

| Команда | Опис |
|---------|------|
| `/broadcast` | Запустити розсилку |
| `/scheduled` | Переглянути заплановані розсилки |
| `/cancel` | Скасувати поточну дію |
| `/help` | Інструкція |

## Налаштування

Скопіюй `.env.example` в `.env` і заповни:

```env
BOT_TOKEN=your_bot_token_here
ADMIN_CHAT_ID=123456789
GROUP_CHAT_ID=-1001234567890
```

- `BOT_TOKEN` — токен від [@BotFather](https://t.me/BotFather)
- `ADMIN_CHAT_ID` — твій Telegram ID (можна дізнатись через [@userinfobot](https://t.me/userinfobot))
- `GROUP_CHAT_ID` — ID групи, куди відправляти (від'ємне число)

## Запуск через Docker

```bash
docker compose up -d --build
```

## Локальна розробка

```bash
npm install
npm run dev
```

> Час завжди рахується по Києву (`Europe/Kyiv`).
